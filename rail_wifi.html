Before You Start
Wi-Fi is a radio signal. Check these things before cycling the rail.
Check nearby rails first
If 2+ rails in the same area have no Wi-Fi, it's the access point — not the rails. Tell your supervisor. Don't cycle anything.
Metal blocks signal
Steel shelving and wire mesh absorb Wi-Fi. Weak signal on a metal-heavy aisle is normal. No signal is still worth cycling.
Water kills signal
Freezer cases and wet product drop Wi-Fi hard. Note it if this rail is near a refrigerated zone.
No power = no Wi-Fi
A dead rail has no radio. Check power first — then come back here if Wi-Fi is still down.

Power Cycle
Complete all 5 steps in order. Wait 60 sec after reconnecting before re-testing.
1
Open Both Access Panels
One panel per battery. Open both before doing anything else.
2
Unplug Both Batteries
Pinch the connector and pull straight out. Do both. Do not pull by the wires.
3
Wait 30 Seconds
Let the rail fully discharge before reconnecting.
4
Plug Both Batteries Back In
Push each connector in firmly until it clicks. LEDs should flash to confirm power.
5
Close Both Panels
Press each cover until it clicks flush. Wait 60 seconds, then re-test.
Power Cycle Complete
Re-test the rail to check if Wi-Fi has recovered. If the issue continues, go back to Rail Detail — you can swap the rail from there.
