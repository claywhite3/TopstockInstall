Before You Start
BLE and Wi-Fi are both radio signals. Check these things before cycling the rail.
Check nearby rails first
If 2+ rails in the same area have no signal, it's the access point — not the rails. Tell your supervisor. Don't cycle anything.
Metal blocks signal
Steel shelving and wire mesh absorb radio. Weak signal on a metal-heavy aisle is normal for both BLE and Wi-Fi. No signal is still worth cycling.
Water kills signal
Freezer cases and wet product drop radio signal on both BLE and Wi-Fi. Note it if this rail is near a refrigerated zone.
No power = no network
A dead rail has no radio. Check power first — then come back here if either BLE or Wi-Fi is still down.
Power Cycle
Complete all 5 steps in order. Wait 60 sec after reconnecting before re-testing.
1
Open Both Access Panels
One panel per battery. Open both before doing anything else.
2
Unplug Both Batteries
Pinch the connector and pull straight out. Do both. Do not pull by the wires.
3
Wait 30 Seconds
Let the rail fully discharge before reconnecting.
4
Plug Both Batteries Back In
Push each connector in firmly until it clicks. LEDs should flash to confirm power.
5
Close Both Panels
Press each cover until it clicks flush. Wait 60 seconds, then re-test.
First cycle didn't resolve it
Radio issues can be environmental. Run the cycle once more before we try a rail swap.
